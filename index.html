Neuroscience and Machine Learning project

Researchers from the University of Cambridge (Cambridge Neuroscience) and the University of Cardiff (School of Computer Science and Informatics) participated in this project. The idea after the estimation of the forward/inverse models (EEG data) is finding a common sensor space. Therefore, the next steps for each subject will be employed:

  • map subject to source space
  • map it back in the 'common sensor space.'
  • GitHub project: https://github.com/RickLicona/3-Layer-BEM-model

The final step is the evaluation of the employed method, using some techniques such as group decoding.

Python Machine Learning Developer at CIDETEC-IPN

Gamma Framework V1.0.

As a member of the Alfa-Beta research group, I transformed the theory of the Mexican Gamma classifier into Python 3.0 code. The Gamma Framework V1.0 with its methods, has the capacity to:

  • Generate descriptive statistics
  • Show the correlation that exists between the features
  • Verify the existence of the missing values
  • Scale and prepare the data for the Gamma classifier
  • Divide the data using a Hold-Out stratified method
  • Generate a confusion matrix after the prediction of the test data
  • Give performance metrics such as accuracy and mean absolute error
  • GitHub project: https://github.com/RickLicona/GammaClassifier
